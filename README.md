The Ice Cream Order Website is a responsive web application developed to provide a smooth and simple online ordering experience for ice cream lovers. The website allows users to browse different flavors, add items to their cart, and place orders easily.

* View a variety of ice cream flavors

* Add items to cart

* Quantity selection

* Automatic total price calculation

* Simple and user-friendly interface

* Fully responsive design
